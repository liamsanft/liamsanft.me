# liamsanft.me

My personal portfolio site.

## Tech stack

- Sveltekit
- TailwindCSS
- Vercel

## Overview

- `src/*` - All app related files
- `lib/*` - Helpful utilities or components
- `static/*` - Static folder
- `tests/*` - Folder containing all tests

## Running locally

```bash
$ git clone https://github.com/liamsanft/liamsanft.me.git
$ cd liamsanft.me
$ pnpm install
$ pnpm dev
```
